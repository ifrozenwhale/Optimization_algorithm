# 单纯形法（对偶单纯形）实现

## 算法实现

在编写单纯形算法求解线性规划问题时，会出现以下几种常见的情况。
1. 没有直观的基可行解。
2. 目标函数检验数非正。

因此，需要讲以上几种算法进行结合。具体算法实现如下。

1. 添加人工变量使得问题变为线性规划的标准形式。
2. 确定基可行解，如果没有，则添加大 M。
3. 输入目标函数和约束方程组成的系数矩阵。
4. 判断检验数正负情况，如果全部非正，则将目标函数的系数矩阵取反，并调用对偶单纯形法
的转轴操作。否则调用单纯形法的转轴操作。
5. 确定入基变量和离基变量后。
6. 进行高斯消元矩阵行变换。
7. 迭代，最终输出结果。

## 输入说明

输入的系数矩阵元素如表所示。

| 目标函数系数 | 目标函数值   |
| ------------ | ------------ |
| 约束方程系数 | 约束方程常数 |

程序提供了两种输入方式。

1. 通过控制台输入。
2. 通过在 main 函数中构造矩阵的方式。
